🚗 JLR Smartcar – Einrichtung

Schritt 1 – Zugangsdaten im Smartcar Dashboard holen

  1. Öffne dashboard.smartcar.com und melde dich an.
  2. Wähle deine App aus.
  3. Kopiere Client ID und Client Secret aus dem Bereich API credentials.
  4. Kopiere den Application Management Token aus dem Bereich Application Configuration.

💡 Die userId wird automatisch über den Management Token aufgelöst – du musst sie nicht manuell suchen.

Schritt 2 – Verbindung testen

Schritt 3 – Config speichern

Trage clientId, clientSecret und managementToken in die Plugin-Config ein und starte Homebridge neu. Die userId wird beim Start automatisch aufgelöst.
